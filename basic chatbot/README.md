# Basic Python Chatbot

A simple rule-based chatbot built with Python.

## Features

- Responds to greetings
- Answers basic questions
- Easy to understand
- Beginner-friendly

## Project Structure

```
basic-chatbot/
│── chatbot.py
│── responses.py
│── requirements.txt
│── README.md
│── .gitignore
```

## Run

```bash
python chatbot.py
```

## Example

```
🤖 Welcome to Basic Chatbot

You: Hello
Bot: Hello! How can I help you?

You: How are you?
Bot: I'm doing great. Thanks for asking!

You: exit
Bot: Goodbye! 👋
```
